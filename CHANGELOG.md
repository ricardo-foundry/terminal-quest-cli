# Changelog

所有版本的变更记录都在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] - 2026-02-14

### 🎉 新增
- 全新的 RPG 等级系统，7个等级从新手到 KIMI 的伙伴
- 18个成就系统，解锁获得 EXP 奖励
- 12个主线任务引导剧情
- 5个迷你游戏：贪吃蛇、猜数字、矩阵雨、乒乓球、Wordle
- NPC 对话系统，与向导 AI 和商人 AI 互动
- 背包系统，收集特殊物品
- 自动存档功能

### 🎮 游戏内容
- 5个可探索区域：家目录、系统核心、世界枢纽、交易站、暗影领域
- 丰富的文件系统，包含隐藏文件和加密信息
- 3个密钥片段等待收集
- 终极秘密等待解锁

### 🔧 技术
- 使用 chalk 实现彩色终端输出
- 使用 figlet 实现 ASCII 艺术标题
- 支持键盘事件处理（方向键、WASD）
- 非 TTY 环境降级显示

### 📝 文档
- 完善的中文 README
- 详细的命令说明
- 游戏目标指引

## [1.0.0] - 2026-02-01

### 🎉 初始版本
- 基础的终端文件系统探索
- ls, cd, cat, pwd 等基本命令
- 简单的隐藏文件发现机制
- 成就系统雏形
