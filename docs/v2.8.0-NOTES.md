# Terminal Quest v2.8.0

The biggest content + polish drop yet. Run it now:

```bash
npx terminal-quest-cli@2.8.0
```

## What's new for players

### Three new quests
- **`cyber-bazaar`** — a summer-only night market with three endings
  (broker_friend / smuggler / fair_buyer). Build affinity with the
  shopkeeper and bring the right items.
- **`forgotten-archive`** — autumn / winter slow-burn arc. Decode old
  files, gain the keeper's trust, and watch a turn-counter ticker
  unlock the final scene.
- **`orbital-station`** — winter endgame. Requires items from the
  other two quests plus all three master-key fragments. Three
  branches: hero, saboteur, operator.

### New Game+
Finished the main story? Boot with `--ng` (or `--new-game-plus`) to
roll the slot into a fresh playthrough that keeps your achievements,
NPC affinities, locales played, lifetime command count, and finished
community arcs. World progression resets, NG+ cycle increments, and
five new NG+ exclusive achievements unlock as you replay.

### Tutorial command
New player? Type `tutorial` in-game for a 12-step, ~5-minute
walkthrough of every pillar (navigation, scan/decode, NPCs, gifts,
seasons, bookmarks, quests, sharing). It only prints guidance — never
auto-executes — so it's safe at any save point.

### Optional text-to-speech
Boot with `--tts` (or toggle in-game with `tts on` / `off` /
`status`). Uses your OS's native voice (`say` on macOS, `espeak` on
Linux, SAPI on Windows) with zero new runtime dependencies. NPC
dialog is the first surface piped through.

### Idle / AFK detection
- Soft prompt at 5 minutes idle.
- Auto-save + warning at 10 minutes — the game stays alive, just
  type any command to resume.

### Hidden goodies
- A new hidden quest, `echo-of-claude`, lives somewhere in
  `/var/log/sessions/`. Try `cat /var/log/sessions/ghost.log`.
  Three branches keyed off your alignment.
- A `ghost_in_the_machine` achievement waits for you on completion.
- Run with `--credits` to see a slow-rolling contributor scroll.

### Polish
- `help` is now alphabetised within each section (Basic / Advanced
  / RPG / Meta / Hidden) for easier scanning.
- Lang line lists every supported locale: `en | zh | zh-tw | ja | es`.
- README adds a "Known limitations" section that catalogs every
  intentional cap (alias chain depth, save bytes, time-advance
  clamp, predicate sandbox, fs.watch best-effort, TTS host
  dependency, folder/id rule).

## What's new for contributors

- **`docs/USER_GUIDE.md`** — full player handbook covering CLI
  flags, every in-game command, quests, relationships, seasons,
  achievements, TTS, and save data.
- **`docs/JOURNEY.md`** — end-to-end timeline of iter-1..iter-21 with
  theme, headline outcome, and running test count per iter.
- **`SECURITY.md`** has a real support-range table and documented
  Node.js matrix.

## Compatibility

- Saves are forward-compatible. New fields (`tutorialSeen`, `ngPlus`,
  `ngCount`, `ngAchievements`, `unlockedQuestIds`, `totalCommands`,
  `totalPlaytimeMs`) default to safe falsy values when missing.
- TTS is opt-in and a no-op on any platform we can't detect, so CI
  / headless environments are unaffected.
- No new runtime dependencies. Still just `chalk`, `figlet`,
  `keypress`. Node.js `>=14`.
